# Classic Game Shuffle

A game shuffle is an event where players go through a set of games in a random order. The twist is that the game you are currently playing can get swapped to one of the other games at any moment. 

## Key Features:
* No Installs - The CGS is a web app meaning that participants only need a reasonably up to date web browser to play. 
* Easy Setup - There is only one keybind menu that is used for all games.
* Synchronized Shuffles - What game participants are playing and for how long is controlled by a separate server program ensuring everyone is playing the same game at the same time.
* Just Game - An initial state is loaded at the start allowing games to skip past any menuing. Infinite life cheat codes let players stay in the game uninterrupted.
